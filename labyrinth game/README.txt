Game info:
Possible number of players(2/3/4) number of cards(12/24) labyrinth size(5/7/9/11)

Game menu:
New game
Load game
Exit game

Whenever during game commands:
Exit application, enter exit
Load saved game, enter load

At the start of players turn commands:
Restore last turn game state, enter undo
Save game state, enter save
